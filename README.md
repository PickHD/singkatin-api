# Url Shortener Public API with Express.js

## Setup

```
npm install --save
```
Following Another Script :

### Lint

```
npm run lint
```

### Test

```
npm run test
```
### Test with reports
```
npm run coverage
```

### Development Mode

```
npm run dev
```
Server will running on port : 8080

### Production Mode

```
npm run start
```
Make sure to migrating table first. Following code :
```
npm run migrate
```
To undo :
```
npm run undo_migrate
```
